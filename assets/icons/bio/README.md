# BioResearch Agent Icons

此目录存放生物科研版本的品牌图标。

## 所需图标

- `app-icon.png` - 应用程序图标（512x512 或更高分辨率）
- `logo.png` - 欢迎页 Logo（建议 96x96 或更高）

## 设计建议

- **主题色**：蓝绿色系 (#2B7A78)
- **风格**：科技感、专业感、学术感
- **元素参考**：DNA双螺旋、显微镜、分子结构、实验器材等生物科研相关图形

## 临时方案

在正式图标设计完成前，可以：
1. 复制 business 目录的图标作为占位符
2. 使用在线工具生成临时图标
3. 委托设计师根据品牌定位设计专属图标
